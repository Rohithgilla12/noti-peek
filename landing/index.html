What it does

Built to disappear into your workflow.

01 · Bundling

Noise collapses into one row.

Multiple comments on the same PR or issue fold into a single entry with an actor stack, event-type summary and unread count. Always a keystroke away from the raw feed, never your default.

02 · Coverage

Four tools, one inbox.

GitHub and Linear on day one. Jira and Bitbucket behind an experimental flag. Consistent UI, consistent shortcuts, consistent mark-as-read semantics across all of them.

03 · Local

Runs on your laptop.

Native Tauri binary, roughly 15 MB. Tokens are stored encrypted on-device; notification bodies never leave your machine. See the privacy statement.

04 · Open source

MIT-licensed. Read every line.

Client, backend and landing page all live in one public repo. Audit the code, fork it, self-host the Worker, or send a pull request — issues and PRs are genuinely welcome.

Questions

Fair to ask.

Is this free? ›

Yes — and it's open source under MIT. Builds are free during beta. Paid tiers may appear later for heavier providers or team features, but the core app will stay open source and self-hostable — existing users get clear notice, nothing swapped under you.

Where are my OAuth tokens stored? ›

In D1 (Cloudflare's SQLite), encrypted at rest, per-device. Your device only ever sees its own connection. No third-party analytics, no token sharing, no token export. Details live in the privacy statement.

Can I run it without the hosted backend? ›

Yes. The backend is one Cloudflare Worker and two D1 migrations; self-hosting takes about ten minutes if you're comfortable with wrangler deploy. The frontend points to any API URL you configure.

What happens when I disconnect a provider? ›

The app revokes the token with the provider (where their API supports it) and deletes the connection row from D1. Nothing kept server-side after disconnect.

Windows or Linux? ›

Not yet. Tauri makes it possible, but the current focus is a polished macOS menubar experience. Watch the repo — that's where it'll show up when it does.
