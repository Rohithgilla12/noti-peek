Questions

Is this free?

The current builds are free while noti-peek is in beta. Paid tiers may appear later for heavier providers or team features — existing users will get clear notice, nothing swapped under you.

Where are my OAuth tokens stored?

In D1 (Cloudflare's SQLite), encrypted at rest, on a per-device token. Your device only ever sees its own connection. No third-party analytics, no token sharing, no token export. Details live in the privacy statement.

Can I run it without the hosted backend?

Yes — the backend is a single Cloudflare Worker and two D1 migrations; self-hosting takes about 10 minutes if you're comfortable with wrangler deploy. The frontend points to any API URL you configure.

What happens when I disconnect a provider?

The app revokes the token with the provider (where their API supports it) and deletes the connection row from D1. Nothing is kept server-side after disconnect.

Windows or Linux?

Not yet. Tauri makes it possible, but the current focus is a polished macOS menubar experience. Watch the repo if you want cross-platform — that's when it'll show up.
